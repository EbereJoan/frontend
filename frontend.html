What is Frontend development?

Frontend development also known as the client-side development is the development of the graphical user interface of a website using HTML, CSS, and Javascript which aids the user in viewing and interacting with the website or application

Why I chose the path of Frontend development

  • Because frontend developers are in high demand by top-tier companies ranging from health-care institutions to educational institutions, financial institutions, etc
  • It's lucrative
  • It helps me keep up with new technologies
  • Frontend development is flexible
